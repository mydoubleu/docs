# Bookstack 설치 (Docker)

https://www.bookstackapp.com/docs/admin/installation/#dockeropen in new windowhttps://github.com/linuxserver/docker-bookstackopen in new window

설치 전에 MariaDB에서 bookstack database/user 생성

mkdir /opt/bookstack
docker run -d \
  --name=bookstack \
  -e PUID=1000 \
  -e PGID=1000 \
  -e APP_URL=https://bookstack.woojuin.net \
  -e DB_HOST=192.168.78.10 \
  -e DB_USER=bookstack \
  -e DB_PASS=bookstack \
  -e DB_DATABASE=bookstack \
  -p 6875:80 \
  -v /opt/bookstack:/config \
  --restart unless-stopped \
  ghcr.io/linuxserver/bookstack
